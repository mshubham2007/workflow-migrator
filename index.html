Workflow Migrator

About

The Workflow Migrator GitHub App automates the process of converting GitLab CI/CD pipelines into GitHub Actions workflows. It simplifies the migration of projects from GitLab to GitHub, making continuous integration setups quicker and more efficient.

Usage

To use this app, install it in your repository, create an issue with the details of the GitLab pipelines you want to migrate, and trigger the app by commenting `/migrate-workflows`. The app will handle the rest, creating pull requests with the converted GitHub Actions workflows.

Installation

Install the Workflow Migrator from the GitHub Apps store.
